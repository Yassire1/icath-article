\subsection{Limitations}

Our study has several limitations. First, we evaluated three open-source TSFMs; extending the benchmark to include closed-source API models (e.g., TimeGPT) or more recently released models would strengthen the analysis. Second, our few-shot experiments used LoRA adaptation; alternative parameter-efficient fine-tuning methods (e.g., adapters, prompt tuning) may yield different results. Third, Chronos and Lag-Llama are inherently univariate, so we average sensor channels before inference, which discards inter-sensor correlations.

\subsection{Future Work}

The gaps identified in this study motivate several directions for future research:

\textbf{Domain-Specific Pretraining:} Distribution shift between web-scale pretraining corpora and industrial sensor data suggests that domain-specific pretraining corpora could significantly improve zero-shot performance.

\textbf{Physics-Informed TSFMs:} Incorporating monotonic degradation priors into TSFM architectures could improve RUL estimation under cross-condition transfer.

\textbf{Multivariate Foundation Models:} All probabilistic TSFMs evaluated here are univariate. A multivariate probabilistic foundation model suitable for the 52-channel Wind SCADA setting remains an open problem.

\subsection{Impact}

Our reproducible empirical evaluation provides practitioners with actionable evidence for deciding whether and how to deploy TSFMs in industrial PdM contexts. The public code release enables the community to extend the benchmark to new models and datasets as they become available.
